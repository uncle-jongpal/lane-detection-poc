## 공고 요약 (확인된 사항)

- **회사**: 주식회사 충청 (`ccrd.co.kr`)

- **연봉**: 8,000만원
- **마감**: 2026-05-15
- **경력**: 경력무관

## 제안 PoC 구성

### 1) 오픈소스 차선 인식 PoC

- **모델 후보**:
  - Ultra-Fast-Lane-Detection (https://github.com/cfzd/Ultra-Fast-Lane-Detection)
  - CLRNet (https://github.com/Turoad/CLRNet) — SOTA급, PyTorch 기반
- **데이터셋**: TuSimple 또는 CULane (공개 도로 차선 데이터셋)
- **테스트**: 영상 입력 → 차선 검출 결과 시각화

### 2) 웹 데모

- FastAPI 백엔드 + 간단한 프론트
- 이미지/영상 업로드 → 차선 검출 결과 오버레이 표시
- 식품표시법 검증 웹(Flask+HTML) 만든 경험과 동일 구조

### 3) 어필 포인트

- "실제 도로 데이터셋으로 테스트해봤고, 정확도 X% 나왔습니다"
- "실제 AUTONG 환경(실외, 흙먼지 등)에서는 추가 튜닝이 필요하겠지만, 기반 파이프라인은 준비돼있습니다"
- "모델 선정 → 데이터 준비 → 학습 → 웹 데모까지 X일 만에 했습니다"
