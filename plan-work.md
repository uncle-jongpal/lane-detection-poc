## 이 플랜의 차별화 포인트

**"도메인 알고 들어옴"**을 명시적으로 보여주는 게 핵심. PoC 데모 자체보다 **분석 깊이**가 평가받음.

### 데모 흐름 (제안)

A. **건설 현장/오프로드 영상 1~2개** 확보 (유튜브에서 한국 공사장 또는 광산 영상 1~2분)

B. 모델 **2개 동시** 추론:
- CLRNet (정확도 우선) — 도시 도로 학습 모델
- Ultra-Fast-Lane-Detection (속도 우선)
- (선택) YOLOPv2 — 차선 + 주행가능영역 + 객체 한 번에

C. **출력 비교 분석**:
- 어디서 검출되고 어디서 실패하는지 정량화
- "포장도로 95%, 비포장 30%, 차선 마킹 없는 구간 5%" 같이 한계 솔직하게 정리

D. **개선 제안 동봉**:
- "공사장 환경 추가 학습 데이터 수집 (어떤 라벨링 필요)"
- "센서 fusion 가능성 (LiDAR + Camera)"
- "임베디드 적용 시 Ultra-Fast 채택 + 양자화 권장"

E. **결과물 형식**:
- GitHub repo (README + GIF + 비교 영상)
- 8~12쪽 PDF (분석 + 한계 + 다음 단계 제안)
- jupyter notebook (재현 가능)
- 웹 데모는 부가 — 시간 남으면 추가, 없으면 생략
