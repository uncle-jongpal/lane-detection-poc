## 핵심 사업 (이게 PoC 방향을 가른다)

CCRD 는 **자율주행 자동차 회사가 아닙니다**. 정확히는 다음 세 가지 축:

1. **도로교통안전시설물 제조·시공** — 가드레일, 표지판, 시선유도봉, 차선 도색 같은 도로변 인프라
2. **자동천공기** — 도로변에 표지판/가드레일 기둥을 박기 위해 노면·갓길을 자동으로 천공하는 장비
3. **소형건설시공로봇** — 위 시공을 사람 없이 자동화하는 소형 로봇

**즉, 회사가 다루는 환경은 "차량이 달리는 차선 위" 가 아니라 "차선 옆 갓길/도로 경계" 입니다.**

## PoC 방향 재정렬 (plan-work 보강)

plan-doc / plan-work 가 가정한 "차선 검출" 만 가지고는 도메인 매칭이 60% 정도. 이것만으로도 평가자 인상은 plan-doc 보다 좋지만, **CCRD 실제 비즈니스에 정조준** 하려면 다음 축 추가:

### 추가 축 1: 차선 + 도로 가장자리 동시 검출

- 작업 차량/로봇이 **차선과 갓길 사이** 좁은 영역에서 움직이므로 두 경계 모두 정밀하게.
- 일반 차선 검출 모델은 차선만 봄. **road edge** 까지 같이 잡는 모델 (YOLOPv2, BEV 기반) 비교 가치 큼.

### 추가 축 2: 시선유도봉·가드레일·표지판 detection

- 회사가 **시공·교체 대상으로 다루는 객체** 자체를 인식. "이 모델은 차선만 보지만 우리는 가드레일도 같이 봐야 함" 같은 분석.
- 간단한 YOLOv8 으로 5분이면 추가 가능. 기존 모델 부족함을 보여줄 무기가 됨.

### 추가 축 3: 차선 도색이 흐려진 구간 검출

- 회사가 "흐려진 차선을 다시 도색" 하는 시공도 한다면 (확인 안 됨, 가드레일이 더 메인일 수도) 이 케이스에서 **차선 검출 실패율** 정량화는 강력한 어필.
- "기존 모델은 신선한 차선만 잘 검출, 흐려진 차선에서는 X% 실패. 시공 대상 자동 식별에는 따라서 Y 방식이 적합" 식 분석.

## 정조준 PoC 흐름 (제안)

원래 plan-work 에 위 3축을 얹어서:

1. **공사장/도로변 작업 영상** 1~2개 (유튜브, 드론 영상 등)
2. **다중 모델 동시 추론**:
   - CLRNet — 차선 검출 SOTA
   - Ultra-Fast-Lane-Detection — 속도 우선 (임베디드 적합성)
   - YOLOPv2 — 차선 + 주행가능영역 + 객체 (도로 인프라용 멀티태스크)
   - YOLOv8 (선택) — 가드레일/표지판/시선유도봉 detection 추가
3. **CCRD 시공 시나리오 정량화**:
   - 정상 차선 검출률 / 흐려진 차선 검출률
   - 갓길·road edge 검출 가능 여부
   - 도로 인프라 객체 검출률
   - FPS (임베디드 컨트롤러 적합성)
4. **개선 제안**:
   - "회사 보유 시공 영상으로 fine-tuning 권장 (라벨링 가이드 첨부)"
   - "센서 fusion (Camera + IMU + RTK GPS) 시공 cm 정확도 확보"
   - "임베디드 적용 시 Ultra-Fast 양자화 + ONNX Runtime"

## 어필 메시지 (지원서·면접용 1~2 문장)

> "CCRD 가 실제로 다루는 도로변 시공 시나리오를 가정하고, 차선·road edge·도로 인프라 객체를 동시에 인식할 수 있는지 SOTA 모델 3종을 비교했습니다. 도시 차선 데이터셋 기반 모델은 흐려진 차선/비포장 환경에서 한계가 명확하며, 회사 보유 시공 영상으로 fine-tuning + 센서 fusion 이 cm 정확도 확보의 빠른 경로라고 봅니다."

## 정보 갭 (확인 못 한 것)

- ccrd.co.kr 자체 웹사이트 — User-Agent 차단인지 self-signed cert 인지로 자동 fetch 실패. **사람이 브라우저로 직접 들어가서** "주력 제품 이미지·영상" 이 무엇인지 확인 필요.
- 채용 공고의 **우대 기술 스택** — saramin 페이지에 채용 중이라고만 나옴. 공고 본문 보면 "PyTorch / OpenCV / 임베디드" 같은 키워드로 plan 추가 보강 가능.
- 회사가 **차선 도색** 도 하는지, **가드레일 시공** 만 메인인지 — 첫 번째면 차선 검출이 정확히 매치, 두 번째면 객체 검출 비중 키워야 함.
